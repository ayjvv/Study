 <isNaN():>
        ㄴ NaN을 확인하는 함수. NaN 맞으면 true, 아니면 false.
        ㄴ NaN은 비교함수로 판별하지 않음
        ㄴ isNaN('문자열'); true 반환하기 때문에 문자임을 인식할 수 있음
        ㄴ isNaN(123); false 반환하면 숫자임을 인식할 수 있음


< 객체 Object>
        객체, 함수, 배열, 정규 표현식 등 변형 가능한 데이터들의 집합
        객체 종류: 코어(내장) 객체 (Array, Math, Date, Number, String), DOM
        객체, BOM 객체...

        기본구조
        {
          "키(key)" : 값(value), <- 속성(property)
          키(key) : 값(value),
          키(key) : 값(value),
          키(key)
        }

        key는 문자형, value는 모든 자료형(함수, 배열, 객체, 변수, 문자열, null...)이 가능

        객체 표현식 const obj = {}
        함수 표현식 const fn = funtion () {}
