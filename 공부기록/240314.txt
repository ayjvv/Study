*css

em: 1em기준 16px
부모의 현재 1em이 자식의 1em

px-em변환 하는 사이트: https://pxtoem.com.au/

rem: root, 즉 최상위 요소(html)의 font-size를 기준으로 설정
vw,vh: 상대 단위인 %는 부모를 기준으로 값을 정하지만, 
vw vh는 부모와는 상관없이 뷰포트를 기준으로 값을 정한다.

display: flex; -  뷰포트나 요소의 크기가 동적으로 변할 때 사용
플렉스 아이템의 가로 세로 정렬 기준 설정. 즉 주축의 방향 설정


flex-direction: row; /* 기본 값 주축은 가로(왼->오른) 교차축은 세로(위 -> 아래) */
flex-direction: column; /* 주축이 세로(위->아래) 교차축(왼-> 오른 )*/
flex-direction: row-reverse; /* 주축은 역방향(오른->왼) */
flex-direction: column-reverse; /* 주축은 역방향(아래->위) */
